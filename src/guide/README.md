## Features

- **Ultra Lightweight** - Memory footprint under 10MB
- **Multi-Channel Support** - Telegram, Discord, Slack, DingTalk, Feishu, WeCom, LINE, QQ and more
- **AI Powered** - 95% of core code generated by AI Agent
- **Truly Portable** - Single binary, supports RISC-V, ARM64 and x86_64
